
## Development

### Installation
```bash
$ git clone https://github.com/AbdelhamidLarachi/react-native-boilerplate && cd react-native-boilerplate
$ yarn install
```

### iOS Simulator
Development with an iOS simulator:
```bash
$ yarn ios
```

### Android Emulator
```bash
$ yarn android
```

### Real Device
```bash
$ yarn start
```
*Then follow instructions in the new tab in your web-browser.*
